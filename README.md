# Shadertoy with React & WebGL

+ Write program with OpenGL Shading Language (GLSL)
+ Use textures with custom names

## Available uniforms:
+ Time in milliseconds from start (`uniform float uTime`)
+ Mouse position on canvas (`uniform vec2 uMouse`)
